# 데이터 파일 디렉토리

이 디렉토리는 데이터 수집 스크립트에서 사용하는 참조 데이터를 저장합니다.

## 필요한 파일

### subway_stations.csv

지하철역 위치 데이터 (교통 접근성 계산용)

**다운로드:**
- 공공데이터포털: https://www.data.go.kr/data/15099316/fileData.do
- 서울교통공사 1~8호선 역사좌표(위경도) 정보

**파일 형식:**
```csv
역명,위도,경도
강남역,37.4980,127.0276
서초역,37.4830,127.0330
...
```

**컬럼 설명:**
- `역명`: 지하철역 이름
- `위도`: 위도 좌표 (소수점 6자리)
- `경도`: 경도 좌표 (소수점 6자리)

**사용:**
`collect_footfall_data.py` 스크립트에서 `--include-transit` 옵션 사용 시 필요합니다.

---

### schools.csv

학교 위치 데이터 (POI 밀도 계산용)

**다운로드:**
- 공공데이터포털: https://www.data.go.kr/data/15021148/standard.do
- 전국초중등학교위치표준데이터

**파일 형식:**
```csv
학교명,위도,경도
서울초등학교,37.4980,127.0276
...
```

**컬럼 설명:**
- `학교명`: 학교 이름
- `위도`: 위도 좌표
- `경도`: 경도 좌표

---

### hospitals.csv

병원 위치 데이터 (POI 밀도 계산용)

**다운로드:**
- 공공데이터포털: https://www.data.go.kr/data/15000736/openapi.do
- 국립중앙의료원 전국 병·의원찾기서비스
- 또는 표준데이터: https://www.data.go.kr/data/15096293/standard.do

**파일 형식:**
```csv
기관명,위도,경도
서울대학교병원,37.5665,126.9780
...
```

**컬럼 설명:**
- `기관명`: 병원/의원 이름
- `위도`: 위도 좌표
- `경도`: 경도 좌표

---
