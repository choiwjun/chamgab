## 시작하기

### 회원가입

1. **홈페이지 접속**: [https://chamgab.com](https://chamgab.com)
2. **회원가입 버튼 클릭**: 우측 상단 "회원가입" 버튼
3. **소셜 로그인 또는 이메일 가입**:
   - **카카오톡**: 카카오 계정으로 간편 로그인
   - **이메일**: 이메일과 비밀번호로 회원가입

### 비회원 이용

비회원도 제한적으로 서비스를 이용할 수 있습니다:

- **매물 검색 및 조회**: 무제한
- **참값 분석 조회**: 하루 3회 제한
- **상권 분석**: 조회 불가 (회원 전용)
- **통합 대시보드**: 조회 불가 (회원 전용)

> 💡 **팁**: 회원 가입 시 모든 기능을 무제한으로 이용하실 수 있습니다.

## 매물 검색

### 기본 검색

1. **홈 화면 검색창 사용**:

   ```
   예: "강남구 역삼동", "래미안", "84㎡"
   ```

2. **자동완성 기능**:
   - 지역명 입력 시 관련 지역 추천
   - 아파트 브랜드 입력 시 매물 추천

3. **검색 결과 확인**:
   - 리스트 뷰: 매물 목록으로 보기
   - 지도 뷰: 지도에서 위치 확인

### 고급 필터링

**필터 옵션**:

- **가격 범위**: 최소/최대 가격 설정
- **면적 범위**: 원하는 평수 지정
- **건축년도**: 신축/준신축/일반
- **층수**: 저층/중층/고층
- **브랜드**: 래미안, 자이, 푸르지오 등

**적용 방법**:

1. 검색 결과 페이지 좌측 "필터" 패널
2. 원하는 조건 선택
3. "적용" 버튼 클릭

### 지도 검색

1. **지도 뷰 전환**: 우측 상단 "지도" 버튼
2. **지역 탐색**: 지도를 드래그하여 원하는 지역 확인
3. **마커 클릭**: 매물 마커 클릭 시 간단 정보 표시
4. **상세 보기**: 프리뷰 카드에서 "상세보기" 클릭

## 참값 분석

### 참값이란?

**참값 (Chamgab)**은 AI가 분석한 아파트의 적정 가격입니다.

- XGBoost 머신러닝 모델 기반
- SHAP을 통한 가격 결정 요인 분석
- 실거래가 데이터 학습

### 참값 분석 보기

1. **매물 상세 페이지 접속**
2. **"참값 분석" 섹션 확인**
3. **주요 정보**:
   - **참값 가격**: AI가 예측한 적정 가격
   - **시장 가격**: 현재 매물 가격
   - **가격 차이**: 참값 대비 고평가/저평가 여부
   - **투자 점수**: 0-100점 (높을수록 좋음)

### 가격 결정 요인

**SHAP 분석 그래프**:

- **지역 (약 74%)**: 가장 큰 영향 요인
- **면적 (약 18%)**: 두 번째 영향 요인
- **브랜드 (약 3%)**: 프리미엄 브랜드 가치
- **층수, 건축연도** 등

**활용법**:

```
예시:
- 지역 기여도: +1억 8,500만원
- 면적 기여도: +4,625만원
- 브랜드 기여도: +750만원
→ 총 예측가: 12억 5,000만원
```

### 투자 지표

- **ROI (1년)**: 예상 1년 수익률
- **ROI (3년)**: 예상 3년 수익률
- **전세가율**: 매매가 대비 전세가 비율 (70% 이상 주의)
- **유동성 점수**: 매물 회전율 기반 (높을수록 빨리 매도 가능)

## 상권 분석

### 상권 분석 시작하기

1. **메뉴에서 "상권 분석" 클릭**
2. **분석할 상권 선택**:
   - 지역 선택: 서울, 경기 등 광역시·도
   - 상권 선택: 역삼역, 강남역 등 특정 상권

3. **업종 선택**:
   - 한식 음식점
   - 카페
   - 편의점
   - 기타 서비스업

4. **"분석 시작" 버튼 클릭**

### 성공 확률 분석

**AI 창업 성공 예측**:

- **성공 확률**: 0-100% (75% 이상 "높음")
- **등급**: 매우 높음 / 높음 / 보통 / 낮음
- **근거**:
  - 유동인구 점수
  - 매출 밀도
  - 경쟁 밀집도
  - 성장률

**해석 예시**:

```
성공 확률: 75.5%
등급: 높음

근거:
✓ 높은 유동인구 (88점)
✓ 적절한 경쟁 밀집도 (중간)
✓ 상권 성장률 5.2% (양호)
```

### 시간대별 유동인구

**그래프 분석**:

- **피크 타임**: 12-13시, 18-19시 (점심/저녁 시간)
- **오프피크**: 새벽 3-5시
- **운영 시간 최적화**: 피크 타임 집중 운영

**활용법**:

- 음식점: 점심/저녁 피크 타임 대비
- 카페: 오후 시간대 집중
- 편의점: 24시간 운영 vs 시간제 운영 결정

### 연령대별 고객 분석

**인구통계학적 분석**:

- **주요 고객층**: 20대, 30대, 40대 등
- **성별 분포**: 남성/여성 비율
- **지배적 세그먼트**: "30대 여성" 등

**마케팅 활용**:

```
주요 고객층: 30대 여성 (32.8%)

권장 전략:
→ SNS 마케팅 (인스타그램)
→ 건강식 메뉴 강화
→ 포토존 설치
```

### 주말/평일 비교

**패턴 분석**:

- **평일 중심 상권**: 오피스 밀집 지역
  - 평일 > 주말 (유동인구, 매출)
  - 점심 시간 집중

- **주말 중심 상권**: 주거/관광 지역
  - 주말 > 평일
  - 오후~저녁 시간 집중

**운영 전략**:

```
평일 중심 상권:
→ 주말 휴무 가능
→ 평일 점심 메뉴 강화

주말 중심 상권:
→ 주말 인력 보강
→ 가족 단위 메뉴 준비
```

### AI 업종 추천

**추천 업종 TOP 5**:

1. **업종명**: 한식 음식점
   - **매칭 점수**: 92.5/100
   - **성공 확률**: 78.5%
   - **추천 이유**:
     - 높은 점심 유동인구 (직장인 밀집)
     - 경쟁 밀집도 낮음
   - **예상 월 매출**: 1,500만원

2. **업종명**: 카페
   - **매칭 점수**: 88.3/100
   - ...

**활용법**:

- 업종 변경 고려 시 참고
- 신규 창업 시 아이템 선정
- 경쟁 업종 대비 우위 확인

## 통합 대시보드

### 통합 분석이란?

**아파트 투자 + 상권 분석 + 생활 편의성**을 하나의 화면에서 확인

**접근 방법**:

1. 매물 상세 페이지
2. 하단 "통합 분석" 탭 클릭

### 통합 투자 점수

**계산 방식**:

```
통합 점수 = (아파트 투자 점수 × 60%) + (생활 편의성 점수 × 40%)

예시:
아파트 투자 점수: 85점
생활 편의성 점수: 80점

통합 점수 = (85 × 0.6) + (80 × 0.4) = 83점
```

**등급 기준**:

- **90점 이상**: 최우수 (★★★★★)
- **80-89점**: 우수 (★★★★)
- **70-79점**: 양호 (★★★)
- **60-69점**: 보통 (★★)
- **60점 미만**: 주의 (★)

### 아파트 투자 분석

**핵심 지표**:

- **투자 점수**: 0-100점
- **1년 ROI**: 예상 수익률
- **3년 ROI**: 중장기 수익률
- **전세가율**: 매매가 대비 전세가 비율
- **유동성 점수**: 거래 활성도

### 생활 편의성 분석

**세부 점수**:

- **교통 점수 (90점)**: 지하철역 2개, 버스정류장 8개
- **상업 시설 (85점)**: 마트, 편의점, 쇼핑몰 접근성
- **교육 시설 (75점)**: 초중고 학교, 학원가
- **의료 시설 (80점)**: 병원, 약국
- **공원/여가 (75점)**: 공원, 체육시설

**시설 개수**:

- 지하철역: 2개 (도보 5분 이내)
- 버스정류장: 8개
- 초중고 학교: 5개
- 병원: 3개
- 공원: 4개

### 근처 상권 정보

**상권 카드**:

```
[역삼역 상권]
거리: 300m (도보 4분)
성공 확률: 75.5%
평균 월 매출: 1,500만원
유동인구 점수: 88점
```

**활용법**:

- 투자 후 임대 수익 가능성
- 생활 편의성 (식당, 카페, 편의시설)
- 재산가치 상승 가능성

## 알림 센터

### 알림 설정

1. **관심 매물 추가**:
   - 매물 상세 페이지에서 "관심 매물 추가" 클릭

2. **자동 알림 받기**:
   - 가격 변동 알림
   - 상권 성장 알림
   - 투자 기회 알림

### 알림 유형

#### 1. 가격 변동 알림

**언제 받나요?**

- 관심 매물 가격이 5% 이상 변동 시

**내용 예시**:

```
⚠️ 가격 변동 알림

래미안 역삼 84㎡ 가격이 5% 상승했습니다.

이전 가격: 12억원
현재 가격: 12억 6,000만원
변동률: +5.0%
```

#### 2. 상권 성장 알림

**언제 받나요?**

- 근처 상권 매출이 10% 이상 성장 시

**내용 예시**:

```
ℹ️ 상권 성장 알림

역삼역 상권이 지난 분기 대비 10% 성장했습니다.

이전 월평균 매출: 1,350만원
현재 월평균 매출: 1,485만원
성장률: +10.0%

→ 주변 부동산 가치 상승 예상
```

#### 3. 투자 기회 알림

**언제 받나요?**

- 참값 대비 저평가 매물 발견 시
- 추천 투자 점수 80점 이상 매물 등록 시

**내용 예시**:

```
🔥 투자 기회 알림

관심 지역에 투자 기회가 발생했습니다!

매물: 자이 삼성 84㎡
참값: 13억원
시장가: 12억 2,000만원
할인율: -6.2%
투자 점수: 88점

→ 지금 확인하기
```

### 알림 필터링

**필터 옵션**:

- **전체**: 모든 알림
- **가격 변동**: 가격 변동만
- **상권 성장**: 상권 관련만
- **투자 기회**: 투자 기회만

**심각도 필터**:

- **전체**: 모든 심각도
- **일반 (ℹ️)**: 정보성 알림
- **주의 (⚠️)**: 중요 알림
- **긴급 (🔥)**: 즉시 확인 필요

### 알림 관리

1. **알림 읽음 처리**: 알림 클릭 시 자동 읽음 처리
2. **알림 삭제**: 스와이프하여 삭제
3. **모두 읽음 처리**: 우측 상단 "모두 읽음" 버튼
4. **새로고침**: 수동 새로고침 버튼 (자동 30초마다 갱신)

## 리포트 생성

### PDF 리포트란?

**종합 투자 분석 리포트**를 PDF로 다운로드하여 오프라인에서도 확인 가능

**포함 내용**:

1. **아파트 분석**: 참값, 투자 점수, ROI
2. **상권 분석**: 근처 상권 성공 확률, 업종 추천
3. **통합 분석**: 통합 투자 점수, 생활 편의성
4. **리스크 분석**: 시장 리스크, 금리 리스크, 유동성 리스크

### 리포트 생성 방법

1. **매물 상세 페이지 하단 "리포트 생성" 버튼**
2. **포함할 섹션 선택**:
   - ✅ 아파트 분석 (필수)
   - ✅ 상권 분석 (선택)
   - ✅ 통합 분석 (선택)
   - ✅ 리스크 분석 (선택)

3. **"PDF 리포트 생성" 버튼 클릭**
4. **생성 완료 대기 (약 10-30초)**
5. **다운로드 또는 공유**

### 리포트 공유

**공유 방법**:

1. **링크 복사**: 클립보드에 공유 링크 복사
2. **카카오톡 공유**: 카카오톡으로 직접 공유 (향후 지원)
3. **QR 코드**: QR 코드 생성하여 공유 (향후 지원)

**유효 기간**:

- 생성 후 7일간 유효
- 만료 후 재생성 필요

**활용 사례**:

- 가족과 투자 결정 논의
- 부동산 중개업자와 상담
- 금융기관 대출 심사 자료
- 개인 투자 포트폴리오 관리

## 관심 매물 관리

### 관심 매물 추가

1. **매물 상세 페이지에서 "♡ 관심 매물 추가"** 클릭
2. **메모 추가 (선택)**:
   ```
   예: "2순위 투자 대상", "부모님 집 구매 후보"
   ```
3. **저장**

### 관심 매물 목록

**마이페이지 → 관심 매물**:

- 추가한 모든 관심 매물 확인
- 가격 변동 추적
- 메모 수정
- 비교하기 기능

### 매물 비교

**최대 3개 매물 동시 비교**:

1. **관심 매물 목록에서 2-3개 선택**
2. **"비교하기" 버튼 클릭**
3. **비교 테이블 확인**:

## FAQ

### 일반

**Q: 참값은 얼마나 정확한가요?**
A: XGBoost 모델 기준 MAPE 5.50%, R² 0.9917로 높은 정확도를 보입니다. 다만, 실제 거래가는 시장 상황, 개별 협상 등에 따라 달라질 수 있습니다.

**Q: 데이터는 얼마나 자주 업데이트되나요?**
A:

- 매물 정보: 매일 업데이트
- 실거래가: 주 1회 (정부 공개 주기)
- 상권 데이터: 분기별 업데이트

**Q: 비회원도 사용할 수 있나요?**
A: 네, 다만 일부 기능이 제한됩니다:

- 참값 분석: 하루 3회 제한
- 상권 분석: 회원 전용
- 통합 대시보드: 회원 전용
- 알림: 회원 전용

### 참값 분석

**Q: 참값이 시장가보다 높으면 어떻게 해야 하나요?**
A: 참값 > 시장가는 저평가 상태로, 투자 기회일 수 있습니다. 다만, 다른 요인(유동성, 리스크 등)도 함께 고려하세요.

**Q: 전세가율이 높으면 위험한가요?**
A: 일반적으로 전세가율 70% 이상은 주의가 필요합니다. 전세가 하락 시 갭투자 리스크가 있습니다.

**Q: ROI 예측은 어떻게 계산되나요?**
A: 과거 5년 간 해당 지역의 평균 상승률과 현재 시장 트렌드를 기반으로 예측합니다.

### 상권 분석

**Q: 성공 확률은 무엇을 의미하나요?**
A: 해당 업종이 해당 상권에서 3년 이상 생존할 확률을 AI가 예측한 값입니다.

**Q: 상권 데이터는 어디서 가져오나요?**
A: 서울시 우리마을가게 상권분석서비스, 소상공인시장진흥공단 상권정보 등 공공데이터를 활용합니다.

**Q: 예상 월 매출은 정확한가요?**
A: 해당 상권의 동종 업종 평균 매출을 기반으로 한 참고치입니다. 실제 매출은 개별 점포의 운영 능력에 따라 크게 달라질 수 있습니다.

### 통합 분석

**Q: 아파트 60%, 편의성 40% 가중치는 왜 이렇게 설정했나요?**
A: 부동산 투자의 주목적이 자산 가치이지만, 실거주 시 편의성도 중요하다는 점을 고려한 균형잡힌 비율입니다.

**Q: 가중치를 변경할 수 있나요?**
A: 현재는 고정 가중치이며, 향후 업데이트에서 사용자 지정 가중치 기능을 추가할 예정입니다.

### 알림

**Q: 알림을 끌 수 있나요?**
A: 마이페이지 → 설정 → 알림 설정에서 알림 유형별로 on/off 가능합니다.

**Q: 푸시 알림도 받을 수 있나요?**
A: 현재는 웹 알림만 지원하며, 향후 모바일 앱 출시 시 푸시 알림도 지원할 예정입니다.

### 리포트

**Q: 리포트는 무료인가요?**
A: 회원 가입 시 월 5회까지 무료, 이후 건당 3,000원입니다.

**Q: 리포트를 인쇄할 수 있나요?**
A: PDF 다운로드 후 자유롭게 인쇄 가능합니다.

**Q: 만료된 리포트를 다시 받을 수 있나요?**
A: 재생성이 필요하며, 무료 횟수가 남아있으면 무료로 재생성 가능합니다.
