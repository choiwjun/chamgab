## 2. 문제 정의

### 현재 시장의 문제

| 기존 서비스 | 문제점 |
|-------------|--------|
| KB시세, 호갱노노 | "이 가격이 맞는지" 근거 없음 |
| 직방, 네이버 부동산 | 단순 매물 나열, 가치 판단 불가 |
| 감정평가 | 비용 높고 시간 오래 걸림 |

### 사용자의 Pain Point

1. "이 가격이 적정한지 모르겠어요"
2. "시세 대비 싼 건지, 이유가 있어서 싼 건지 구분 안 돼요"
3. "가격 형성 요인을 알 수 없어요"

---

## 3. 솔루션

### 핵심 가치 제안

| 기존 서비스 | 참값(Chamgab) |
|-------------|---------------|
| "이 부동산 시세는 8억입니다" | "이 부동산 참값은 7.8~8.2억이며, 주요 가격 요인은..." |
| 단일 숫자 | 가격 범위 + 신뢰도 |
| 근거 없음 | 영향 요인 Top-N + 유사 거래 비교 |

### 차별화 포인트

1. **"왜"를 설명**: 가격 형성 요인 Top 10 제공
2. **신뢰도 표시**: 예측의 불확실성을 투명하게 공개
3. **유사 거래 비교**: 실제 거래 데이터 기반 검증

## 5. 핵심 기능

### F1: 참값 분석 (Core Feature)

```
입력: 매물 정보 (유형, 위치, 면적 등)
처리: AVM 모델 + SHAP 분석
출력:
  - 참값: 25억 원
  - 범위: 24.5억 ~ 25.5억
  - 신뢰도: 94%
  - 가격 요인 Top 10
```

### F2: 가격 영향 요인 분석

| 순위 | 요인 | 기여도 |
|------|------|--------|
| 1 | 지하철역 거리 (도보 3분) | +2.1억 |
| 2 | 건축년도 (2019년) | +1.8억 |
| 3 | 브랜드 (래미안) | +1.5억 |

### F3: 유사 거래 비교

- 반경 1km 내 유사 조건 거래
- 면적 ±10%, 년식 ±5년
- 유사도 점수 제공
