# User Flow

## 참값(Chamgab) 사용자 흐름

---

## 1. 핵심 사용자 여정

### Journey 1: 첫 방문 → 참값 확인

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│   홈    │ → │  검색   │ → │ 매물상세 │ → │ 참값분석│
│         │    │         │    │         │    │ (로그인)│
└─────────┘    └─────────┘    └─────────┘    └─────────┘
     │              │              │              │
     │              │              │              │
   검색어         필터적용       상세보기       회원가입
   입력          지역/가격       참값확인       /로그인
```

### Journey 2: 매물 비교 → 결정

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 관심매물 │ → │ 비교하기 │ → │ 리포트  │ → │  결정   │
│  저장   │    │         │    │ 다운로드│    │         │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
```

### Journey 3: 프리미엄 전환

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 무료한도 │ → │ 업그레이드│ → │  결제   │ → │ 프리미엄│
│  도달   │    │  모달   │    │         │    │  사용   │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
```

## 2. 화면별 상세 흐름

### 2.1 홈 → 검색

```
[홈 화면]
    │
    ├── 검색바 입력 (2자 이상)
    │       │
    │       └── 자동완성 표시
    │               │
    │               └── 항목 선택 또는 Enter
    │                       │
    │                       └── [검색 결과]
    │
    ├── 인기 지역 클릭
    │       │
    │       └── [검색 결과] (해당 지역 필터)
    │
    └── 최근 본 매물 클릭
            │
            └── [매물 상세]
```

### 2.2 검색 결과 → 매물 상세

```
[검색 결과]
    │
    ├── 지도/리스트 전환
    │       │
    │       ├── 지도 모드: 클러스터 → 마커 → 상세
    │       └── 리스트 모드: 카드 클릭 → 상세
    │
    ├── 필터 적용
    │       │
    │       ├── 부동산 유형 (아파트, 오피스텔 등)
    │       ├── 지역 (시도/시군구/읍면동)
    │       ├── 가격 범위
    │       ├── 면적 범위
    │       └── 건축년도
    │
    └── 매물 카드 클릭
            │
            └── [매물 상세]
```

### 2.3 매물 상세 화면 흐름

```
[매물 상세]
    │
    ├── 기본 정보 섹션
    │       └── 위치, 면적, 층수, 건축년도
    │
    ├── 참값 분석 섹션 (핵심)
    │       │
    │       ├── 참값 표시 (25억 / 24.5~25.5억)
    │       ├── 신뢰도 (94%)
    │       ├── 가격 요인 Top 5
    │       │       └── [더보기] → 프리미엄 유도
    │       └── 유사 거래 비교
    │               └── [더보기] → 프리미엄 유도
    │
    ├── 입지 분석 섹션
    │       └── 지도 + POI 표시
    │
    ├── 관심 매물 저장
    │       │
    │       └── (비로그인) → [로그인 유도]
    │
    └── 비교하기 추가
            │
            └── [비교 화면]
```

### 2.4 인증 흐름

```
[로그인 필요 액션]
    │
    └── [로그인 모달/페이지]
            │
            ├── 이메일 로그인
            │       └── 비밀번호 입력 → 성공 → 원래 화면
            │
            ├── 소셜 로그인
            │       ├── Google
            │       ├── Kakao
            │       └── Naver
            │
            └── 회원가입
                    │
                    ├── 이메일 입력
                    ├── 비밀번호 설정
                    ├── 약관 동의
                    └── 이메일 인증 → 완료
```

### 2.5 결제 흐름

```
[프리미엄 업그레이드]
    │
    └── [플랜 선택]
            │
            ├── 월간 (₩9,900)
            └── 연간 (₩99,000)
                    │
                    └── [결제 페이지]
                            │
                            ├── 카드 정보 입력
                            └── 결제 완료
                                    │
                                    └── [결제 완료] → 원래 화면
```

## 3. 에러 및 예외 흐름

### 3.1 조회 제한 도달

```
[참값 분석 요청]
    │
    └── (제한 초과)
            │
            └── [업그레이드 모달]
                    │
                    ├── "오늘 무료 분석을 모두 사용했습니다"
                    ├── [프리미엄 시작] 버튼
                    └── [내일 다시 시도] 버튼
```

### 3.2 분석 실패 (데이터 부족)

```
[참값 분석 요청]
    │
    └── (거래 데이터 부족)
            │
            └── [에러 상태]
                    │
                    ├── "거래 데이터가 부족하여 분석이 어렵습니다"
                    └── [유사 매물 보기] 버튼
```

### 3.3 네트워크 오류

```
[API 요청]
    │
    └── (네트워크 오류)
            │
            └── [에러 토스트]
                    │
                    ├── "네트워크 연결을 확인해주세요"
                    └── [다시 시도] 버튼
```
