## 2. 화면별 상세

### S01: 홈 화면

| 항목 | 내용 |
|------|------|
| 경로 | `/` |
| 목적 | 서비스 소개 및 검색 진입점 |
| 인증 | 불필요 |

**구성 요소:**

| 섹션 | 컴포넌트 | 데이터 |
|------|----------|--------|
| Hero | 검색바 + 슬로건 | 인기 검색어 (실시간) |
| 가격 트렌드 | 지역별 카드 | 주간 변동률 API |
| 인기 매물 | 매물 카드 리스트 | 조회수 Top 10 |
| 서비스 소개 | 참값 차별점 3가지 | 정적 |

**검색 자동완성:**
- 2자 이상 입력 시 트리거
- 디바운스: 300ms
- 결과: 지역(3) → 매물(5) → 검색어

### S02: 검색 결과 (리스트)

| 항목 | 내용 |
|------|------|
| 경로 | `/search?region={}&type={}` |
| 목적 | 조건에 맞는 매물 리스트 |
| 인증 | 불필요 |

**구성 요소:**

| 섹션 | 컴포넌트 | 기능 |
|------|----------|------|
| 필터바 | 칩/드롭다운 | 유형, 지역, 가격, 면적 |
| 정렬 | 드롭다운 | 참값순, 최신순, 면적순 |
| 매물 리스트 | 카드 리스트 | 무한 스크롤 (20개씩) |
| 뷰 전환 | 토글 버튼 | 리스트 ↔ 지도 |

**필터 옵션:**

| 필터 | 옵션 |
|------|------|
| 부동산 유형 | 전체, 아파트, 오피스텔, 빌라, 상가, 토지, 빌딩 |
| 지역 | 시도 → 시군구 → 읍면동 (계층) |
| 가격 | 슬라이더 (0 ~ 100억+) |
| 면적 | 슬라이더 (0 ~ 200㎡+) |
| 건축년도 | 5년 이내, 10년 이내, 15년 이내, 전체 |

## 3. 화면 간 연결 (Navigation)

```
홈 (/)
├── 검색바 입력 → 검색 결과 (/search)
├── 인기 지역 클릭 → 검색 결과 (/search?region=)
└── 최근 본 매물 → 매물 상세 (/property/[id])

검색 결과 (/search)
├── 뷰 전환 → 지도 검색 (/search/map)
├── 매물 클릭 → 매물 상세 (/property/[id])
└── 필터 적용 → 검색 결과 갱신

매물 상세 (/property/[id])
├── 관심 저장 → (로그인 필요) → 관심 매물 (/favorites)
├── 비교 추가 → 비교하기 (/compare)
├── 참값 분석 → (제한 초과) → 결제 (/checkout/plans)
└── 유사 거래 클릭 → 해당 매물 상세

하단 네비게이션 (모바일)
├── 홈 → /
├── 검색 → /search
├── 관심 → /favorites
├── 알림 → /notifications
└── MY → /mypage
```
