# 토지 투자 분석 PRD (Product Requirements Document)

> Phase 7: 토지 정보 통합 + 토지×상권 융합 분석

---

## 1. 개요

| 항목        | 내용                                                |
| ----------- | --------------------------------------------------- |
| 기능명      | 토지 투자 분석                                      |
| 슬로건      | "이 땅의 가치를 참값이 알려드립니다"                |
| 전략        | 단계적 접근 (정보 통합 → 융합 분석 → 제한적 적정가) |

## 2. 문제 정의

### 현재 시장의 문제

| 기존 서비스         | 문제점                            |
| ------------------- | --------------------------------- |
| 부동산플래닛        | 토지 데이터는 있지만 AI 분석 없음 |
| 호갱노노            | 아파트 전문, 토지 미지원          |
| 토지이음(eum.go.kr) | 규제 정보만, 투자 관점 분석 없음  |
| 밸류맵              | 시세 추정만, 상권 융합 없음       |

### 사용자의 Pain Point

1. "이 토지가 투자 가치가 있는지 모르겠어요"
2. "공시지가, 실거래가, 용도지역을 한 곳에서 보고 싶어요"
3. "이 땅에 어떤 사업을 하면 좋을지 알고 싶어요"
4. "주변 개발 계획이 토지 가격에 미치는 영향을 알고 싶어요"

## 3. 솔루션 — 3단계 접근

### Stage 1: 토지 정보 통합 (MVP)

> 난이도: ★★☆☆☆ | 기간: 2-3주

공공 데이터를 깔끔하게 정리하여 한 곳에서 보여주는 정보 허브.

```
입력: 지역 선택 또는 지번 검색
출력:
  - 공시지가: 450만원/m²
  - 지목: 대지
  - 용도지역: 제2종일반주거지역
  - 실거래 이력: 최근 5년간 거래 내역
  - 주변 시세: 인근 토지 평균 거래가
```

**데이터 소스:**

- 국토교통부 토지 실거래가 API (`getRTMSDataSvcLandTrade`)
- 개별공시지가 API (VWorld)
- 토지특성정보 API (지목, 용도지역)

### Stage 2: 토지×상권 융합 분석 (핵심 차별화)

> 난이도: ★★★☆☆ | 기간: 3-4주

참값만의 차별점. 기존 상권분석 모델과 토지 데이터를 결합.

```
입력: 토지 위치 (지번 또는 좌표)
출력:
  - 상업적 활용도 점수: 82/100
  - 추천 업종 Top 5:
    1. 카페 (성공확률 74%)
    2. 편의점 (성공확률 71%)
    3. 음식점 (성공확률 68%)
  - 유동인구 분석: 일 평균 3,200명
  - 주변 경쟁 현황: 반경 500m 내 음식점 12개
  - 투자 인사이트: "역세권 + 대학가 + 오피스 밀집 → 카페/음식점 최적"
```

**기존 인프라 재활용:**

- `business_model.pkl` (상권 성공 예측)
- `business_statistics` / `foot_traffic_statistics` (130개 지역)
- `poi_data` (주변 인프라)
- SHAP 기반 요인 분석

### Stage 3: 제한적 토지 적정가 (장기)

> 난이도: ★★★★★ | 기간: 2-3개월

도시 지역, 거래 빈도 높은 토지 유형(대지, 상업용지)에 한정.

```
입력: 토지 정보 (위치, 면적, 지목, 용도지역)
출력:
  - AI 추정가: 8.5억원
  - 추정 범위: 7.8억 ~ 9.2억
  - 신뢰도: 72% (투명 공개)
  - 가격 요인 Top 10 (SHAP)
  - ⚠ "토지 특성상 오차 범위가 클 수 있습니다"
```

## 5. 핵심 기능 상세

### F1: 토지 정보 대시보드 (Stage 1)

#### F1-1: 토지 검색

| 검색 방식 | 설명                            |
| --------- | ------------------------------- |
| 지번 검색 | "서울시 강남구 역삼동 123-4"    |
| 지도 클릭 | 지도에서 필지 선택              |
| 지역 필터 | 시도 > 시군구 > 읍면동 드릴다운 |

#### F1-2: 토지 기본 정보

```
┌─────────────────────────────────────────┐
│ 📍 서울시 강남구 역삼동 123-4           │
│                                         │
│ ┌──────────┬──────────┬──────────┐     │
│ │  지목    │ 용도지역  │ 면적     │     │
│ │  대지    │ 제2종일반 │ 330m²   │     │
│ │          │ 주거지역  │ (100평)  │     │
│ └──────────┴──────────┴──────────┘     │
│                                         │
│ 개별공시지가  12,500,000원/m²           │
│ 공시지가 총액  41.2억원                 │
│                                         │
│ ──────── 실거래 이력 ────────           │
│ 2024.03  42.0억  (127만/m²)            │
│ 2022.11  38.5억  (116만/m²)  ▲ 9.1%   │
│ 2021.05  35.0억  (106만/m²)  ▲ 10.0%  │
└─────────────────────────────────────────┘
```

#### F1-3: 주변 토지 시세

- 반경 500m / 1km 내 동일 지목 토지 거래
- 면적당 단가 비교 (원/m²)
- 시세 추이 차트 (최근 5년)

### F2: 토지×상권 융합 분석 (Stage 2)

#### F2-1: 상업적 활용도 분석

```
이 토지의 상업 활용도 점수: 82/100

📊 분석 근거
├─ 유동인구 밀도: ★★★★☆ (일 3,200명)
├─ 대중교통 접근성: ★★★★★ (역세권 300m)
├─ 주변 배후 수요: ★★★★☆ (오피스 밀집)
├─ 경쟁 포화도: ★★★☆☆ (보통)
└─ 임대 수익 잠재력: ★★★★☆
```

#### F2-2: 추천 업종 분석

기존 `business_model.pkl`을 활용하여 해당 위치에서의 업종별 성공 확률 예측.

```
🏆 추천 업종 Top 5

1위 카페·디저트  성공확률 74%  ████████████████░░░░
   → 20-30대 유동인구 多, 오피스 수요

2위 편의점      성공확률 71%  ███████████████░░░░░
   → 24시간 수요, 역세권 입지

3위 한식당      성공확률 68%  ██████████████░░░░░░
   → 점심 수요 높음, 경쟁 보통

4위 미용실      성공확률 65%  █████████████░░░░░░░
   → 배후 주거지 인접

5위 약국        성공확률 63%  ████████████░░░░░░░░
   → 의료시설 인접, 안정적 수요
```

#### F2-3: SHAP 기반 요인 분석

```
이 위치의 사업 성공에 영향을 미치는 요인:

▲ 긍정 요인                    ▼ 부정 요인
──────────                    ──────────
유동인구 밀도   +15.2%        임대료 수준     -8.3%
대중교통 접근성 +12.8%        경쟁 업체 수    -5.1%
배후 세대 수    +9.4%         주차 환경       -3.2%
소득 수준       +7.1%
```

### F3: 토지 적정가 추정 (Stage 3)

- XGBoost 모델 (토지 전용)
- Features: 면적, 지목, 용도지역, 도로접면, 공시지가, 주변 시세, POI 등
- 대지·상업용지만 대상 (농지·임야 제외)
- 오차 범위 투명 공개

| 지표               | 목표    | 측정 방법 |

| 지표            | 목표      | 측정 방법     |
| --------------- | --------- | ------------- |
| MAPE (대지)     | < 15%     | 실거래가 비교 |
