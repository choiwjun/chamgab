# 토지 투자 분석 화면 설계

> Phase 7: 화면 명세 + 사용자 흐름

---

## 1. 사용자 흐름

```
[메인 페이지]
    │
    ├─ 서비스 선택 "토지 투자 분석" ──┐
    │                                 │
    ▼                                 ▼
[토지 메인 /land] ◄─────────────────────
    │
    ├─ 지역 시세 카드 클릭
    │   └─ /land/search?sigungu=강남구
    │
    ├─ 지번 검색
    │   └─ /land/search?q=역삼동+123
    │
    ▼
[토지 검색 결과 /land/search]
    │
    ├─ 토지 카드 클릭
    │
    ▼
[토지 상세 /land/[pnu]]
    │
    ├─ 기본 정보 탭 (지목, 면적, 공시지가, 실거래)
    ├─ 실거래 이력 탭
    ├─ 주변 시세 탭 (지도)
    │
    ├─ "이 토지에서 창업하기" 버튼
    │
    ▼
[토지×상권 융합 /land/[pnu]/commercial]
    │
    ├─ 상업적 활용도 점수
    ├─ 추천 업종 Top 5
    ├─ SHAP 요인 분석
    ├─ 유동인구 / 경쟁 현황
    │
    └─ "상세 상권분석으로" → /business-analysis?lat=...&lng=...
```

### ServiceSelector에 토지 분석 추가

```
┌──────────────────────────────────────────────┐
│  무엇을 분석할까요?                            │
│                                              │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐    │
│  │ 🏢       │ │ 🏪       │ │ 🗺️       │    │
│  │ 아파트   │ │ 상권     │ │ 토지     │    │
│  │ 적정가   │ │ 분석     │ │ 투자분석 │    │
│  │          │ │          │ │          │    │
│  │ AI가 분석│ │ AI가 예측│ │ 토지 가치│    │
│  │ 한 참값  │ │ 하는 성공│ │ 와 최적  │    │
│  │ 을 확인  │ │ 확률     │ │ 업종 분석│    │
```

## 5. 네비게이션 구조

```
참값 (로고)  아파트  상권분석  토지분석  즐겨찾기  마이페이지
                                ^^^^
                              신규 추가
```

헤더 네비게이션에 "토지분석" 탭 추가 → `/land` 링크.
